┌─────────────────────────────────────────────────────────────────────────┐
│ 步骤1: 创建Parser (2分钟)                                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│  1. 访问 https://app.docparser.com                                      │
│  2. 点击 "Create Document Parser"                                       │
│  3. 填写:                                                                │
│     ┌──────────────────────────────┐                                    │
│     │ Name: CreditPilot_Malaysia   │                                    │
│     │ Type: Bank Statement         │                                    │
│     └──────────────────────────────┘                                    │
│  4. 上传示例PDF:                                                         │
│     docparser_templates/sample_pdfs/1_AMBANK.pdf                        │
│                                                                          │
└─────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────┐
│ 步骤2: 配置6个字段 (2分钟) - 用鼠标点击PDF上的数字                      │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│   PDF预览                           输入字段名                          │
│   ┌────────────────┐               ┌──────────────────┐                │
│   │ Card: xxxx6354 │  ← 点这里 →   │ card_number      │                │
│   │ Date: 01/06/25 │  ← 点这里 →   │ statement_date   │                │
│   │ Prev: 861.17   │  ← 点这里 →   │ previous_balance │                │
│   │ Purch: 8147.54 │  ← 点这里 →   │ total_debit      │                │
│   │ Paym: 0.00     │  ← 点这里 →   │ total_credit     │                │
│   │ Bal: 9008.71   │  ← 点这里 →   │ current_balance  │                │
│   └────────────────┘               └──────────────────┘                │
│                                                                          │
│   提示: 字段名必须完全一致(包括大小写和下划线)                          │
│                                                                          │
└─────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────┐
│ 步骤3: 获取Parser ID (30秒)                                             │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│  1. 点击 Settings → API                                                 │
│  2. 复制 Parser ID (例如: odnzsomkbyeh)                                 │
│  3. 在Replit设置环境变量:                                               │
│     ┌──────────────────────────────────────────┐                        │
│     │ DOCPARSER_PARSER_ID = 你的Parser_ID      │                        │
│     └──────────────────────────────────────────┘                        │
│                                                                          │
└─────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────┐
│ ✅ 完成！现在系统可以:                                                   │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│  ✅ 自动上传PDF → DocParser解析 (95%+准确度)                            │
│  ✅ AI识别银行 (AMBANK/UOB/HSBC...) (99%准确度)                         │
│  ✅ 自动分类归档 (银行_卡号_月份)                                       │
│  ✅ 批量处理42个PDF → 完全无需手动操作                                  │
│                                                                          │
└─────────────────────────────────────────────────────────────────────────┘
