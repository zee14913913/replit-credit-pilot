Welcome to Smart Credit & Loan Manager / 欢迎使用智能信贷与贷款管理系统
Quick Start Guide / 快速开始指南
Admin Features / 管理员功能
  1. Login System / 登录系统:
    Click "Admin" in top right → Use admin email and password
    点击右上角"Admin" → 使用管理员邮箱和密码登录
  2. Upload Credit Card Statements / 上传信用卡账单:
    Navigation bar → "CC Ledger" → Click "Show Upload Form" → Select card → Upload PDF (15 Malaysian banks supported)
    导航栏点击"CC Ledger" → 点击"显示上传表单" → 选择信用卡 → 上传PDF账单（支持15家马来西亚银行）
  3. Upload Savings Statements / 上传储蓄账单:
    Click "Savings" → Upload savings account statements (7 banks supported)
    点击"Savings" → 上传储蓄账户账单（支持7家银行）
  4. Customer Management / 客户管理:
    Admin Dashboard → View all customers, credit cards, and transaction data
    Admin Dashboard → 查看所有客户、信用卡和交易数据
  5. Generate Monthly Reports / 生成月度报表:
    Customer details page → Click "Generate Monthly Report"
    客户详情页 → 点击"生成月度报表"

Customer Features / 客户端功能
  1. Customer Login / 客户登录:
    Click "Login" → Use customer email provided by admin
    点击"Login" → 使用管理员提供的客户邮箱登录
  2. View Statements / 查看账单:
    Customer Portal → View all credit cards and transaction records
    客户Portal → 查看所有信用卡和交易记录
  3. View Reports / 查看月报:
    Customer Dashboard → Download PDF monthly reports
    客户Dashboard → 下载PDF月度报告
  4. Apply for Loans / 申请贷款:
    Click "Loan Matcher" → Upload CTOS report to view matched products
    点击"Loan Matcher" → 上传CTOS报告查看匹配产品
