## 💡 **真实计算方法**

### **方法1: DSR计算法** (所有银行)

```
DSR = (月供总额 / 净收入) × 100%

最大可用月供 = 净收入 × (最高DSR% / 100) - 现有月供

最大贷款额 = 根据PMT公式反算
```

### **方法2: 收入倍数法** (部分银行)

```
最大贷款额 = 月收入 × 倍数

倍数范围:
- 标准: 5-8倍
- CIMB: 8倍
- Bank Rakyat: 10倍
- BSN: 10倍
```

### **方法3: 混合法** (大部分银行实际使用)

```
步骤1: 计算DSR，确认符合银行限额
步骤2: 计算收入倍数限额
步骤3: 取两者中的较低值
步骤4: 考虑信用评分调整
```

## 🎯 **实际案例分析**

### **案例1: 中等收入申请者**

**申请人资料**:
- 月收入 (净): RM 5,000
- 现有月供: RM 1,500
- 申请: 个人贷款
- 期限: 5年

**不同银行的批准结果**:

#### **Maybank** (DSR 70%):
```
最大DSR月供 = RM 5,000 × 70% = RM 3,500
可用月供 = RM 3,500 - RM 1,500 = RM 2,000
假设利率 7% p.a., 5年
最大贷款额 ≈ RM 100,000
```

#### **CIMB** (DSR 75%, 8倍月薪):
```
方法1 (DSR):
最大DSR月供 = RM 5,000 × 75% = RM 3,750
可用月供 = RM 3,750 - RM 1,500 = RM 2,250
假设利率 7% p.a., 5年
最大贷款额 ≈ RM 113,000

方法2 (收入倍数):
最大贷款额 = RM 5,000 × 8 = RM 40,000

实际批准额 = min(RM 113,000, RM 40,000) = RM 40,000
```

#### **RHB** (DSR 60%):
```
最大DSR月供 = RM 5,000 × 60% = RM 3,000
可用月供 = RM 3,000 - RM 1,500 = RM 1,500
假设利率 7% p.a., 5年
最大贷款额 ≈ RM 75,000
```

## 📋 **关键要点**

### ✅ **每家银行的独特之处**:

1. **Maybank**: 低收入DSR限制最严格 (40%)，高收入友好 (70%)
2. **CIMB**: 最高DSR限额 (75%)，8倍月薪，1天批准
3. **Hong Leong**: 最高DSR (80%)，认可100%外国收入
4. **RHB**: DSR较保守 (60%)，只认可45%外国收入
5. **Bank Rakyat**: 最高10倍月薪限额

### ⚠️ **重要注意事项**:

1. **DSR ≠ 贷款批准的唯一标准**
2. **信用评分同样重要** (CTOS/CCRIS)
3. **工作稳定性** (最低6个月-2年)
4. **银行关系** (现有客户可能获得优惠)
5. **总债务负担** (不只是DSR)

## 🔄 **系统实现建议**

### **应该使用的计算逻辑**:

```typescript
function calculateMaxLoan(
  monthlyIncome: number,
  monthlyCommitment: number,
  bank: Bank
) {
  // 1. 计算可用收入
  const netIncome = calculateNetIncome(monthlyIncome, bank);
  
  // 2. 确定DSR限额
  const maxDSR = bank.getMaxDSR(netIncome);
  
  // 3. 计算可用月供
  const maxMonthlyPayment = (netIncome * maxDSR / 100) - monthlyCommitment;
  
  // 4. 根据DSR计算贷款额
  const loanByDSR = calculateLoanFromPayment(
    maxMonthlyPayment, 
    bank.interestRate, 
    bank.tenure
  );
  
  // 5. 根据收入倍数计算
  const loanByMultiplier = monthlyIncome * bank.incomeMultiplier;
  
  // 6. 取较小值
  return Math.min(loanByDSR, loanByMultiplier, bank.maxLoanAmount);
}
```
