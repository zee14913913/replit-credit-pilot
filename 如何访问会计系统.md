# 🏦 如何访问财务会计系统

## ✅ **最简单的方法**

### **步骤1：打开Flask主页（默认页面）**
直接点击Replit的"Webview"按钮即可，或访问：
```
https://[你的项目名].repl.co
```

### **步骤2：点击首页顶部的紫色横幅**
你会看到一个大横幅写着"🏦 财务会计系统 (Accounting System)"

点击其中的按钮：
- **📚 API文档** → 打开Swagger UI（可以在线测试所有API）
- **💼 管理后台** → 打开会计系统管理界面

---

## 🔗 **直接访问链接**

如果按钮不工作，手动访问以下URL：

### **方法A：通过Replit域名**
```
# API文档
https://[你的项目名].repl.co:8000/docs

# 管理后台
https://[你的项目名].repl.co:8000/accounting

# 欢迎页面
https://[你的项目名].repl.co:8000/
```

### **方法B：内部端口（在Replit Shell测试）**
```bash
# 测试API是否正常
curl http://localhost:8000/api/companies/

# 查看会计科目
curl http://localhost:8000/api/companies/1/chart-of-accounts

# 查看试算表
curl "http://localhost:8000/api/reports/trial-balance?company_id=1&year=2024&month=7"
```

- URL: `https://[项目名].repl.co:8000/`

## ❌ **常见错误**

### **错误1: 404 Not Found - 访问/docs时**
**原因**: 你访问的是Flask系统（端口5000）而不是Accounting系统（端口8000）

**解决**:
- 确保URL中有 `:8000`
- 或者从Flask首页点击紫色横幅的按钮

### **错误2: 连接超时**
**原因**: Accounting API workflow未运行

**解决**:
1. 在Replit界面检查"Accounting API" workflow状态
2. 如果未运行，点击重启

## 📱 **访问示例**

### **场景1：我想上传Maybank CSV**
```
1. 打开 https://[项目名].repl.co:8000/accounting
2. 点击"Upload Bank Statement"
3. 选择CSV文件，输入company_id=1, month=2024-08
4. 点击上传
```

### **场景2：我想查看试算表**
```
1. 打开 https://[项目名].repl.co:8000/docs
2. 找到 "GET /api/reports/trial-balance"
3. 点击"Try it out"
4. 输入 company_id=1, year=2024, month=7
5. 点击"Execute"
```
