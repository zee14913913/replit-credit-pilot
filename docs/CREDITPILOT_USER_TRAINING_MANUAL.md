### **功能 2: DSR 计算器** ⭐⭐⭐

#### 什么是 DSR？
DSR（Debt Service Ratio）是**债务偿还比率**，银行用来评估您是否有能力还款。

**公式：**
```
DSR = (现有月供 + 新贷款月供) / 月收入 × 100%
```

**银行标准：**
- DSR ≤ 55%: ✅ **PASS**（银行会批准）
- DSR 55-70%: ⚠️ **BORDERLINE**（需要优化）
- DSR ≥ 70%: ❌ **HIGH**（很难批准）

#### 操作步骤：
1. 在 Loans 页面找到"DSR Calculator"区域
2. 输入以下参数：
   - **月净收入**: 您的薪资（税后）
   - **现有月供**: 所有现有贷款的月供总和
   - **新贷款金额**: 您打算申请的贷款金额
   - **APR 利率**: 产品的年利率
   - **贷款期限**: 贷款年数（如 30 年）
3. 点击"Calculate DSR"按钮
4. 系统立即显示：
   - 新贷款月供
   - 总月供
   - DSR 百分比
   - 批准状态（PASS/BORDERLINE/HIGH）

#### 实际案例：
```
客户: 李先生
月收入: RM 8,500
现有月供: RM 2,450 (房贷 RM 2,100 + 车贷 RM 350)
新贷款: RM 500,000 (30 年, 3.5% APR)

计算结果:
- 新贷款月供: RM 2,245.22
- 总月供: RM 4,695.22
- DSR: 55.2%
- 状态: ✅ PASS（刚好在批准线）

建议:
- 如果增加首付 RM 50,000，DSR 降至 52.8% ✅
- 如果延长至 35 年，DSR 降至 53.5% ✅
```

#### 常见问题：
**Q: 如果我的 DSR 是 60%，怎么办？**  
A: 您有 3 个选择：
1. **增加首付** - 减少贷款金额
2. **延长期限** - 降低月供（但总利息增加）
3. **先还清小额贷款** - 降低现有月供

**Q: DSR 计算包含信用卡吗？**  
A: 不包含。DSR 只计算**贷款月供**，不包含信用卡最低还款额。

## 2.2 CTOS Authorization（CTOS 授权）

### 什么是 CTOS？
CTOS 是马来西亚官方信用报告机构，类似中国的"征信报告"。

### 为什么需要 CTOS？
银行批准贷款时，会查看您的 CTOS 报告，了解：

- 现有贷款和月供

- 还款历史记录

### 操作步骤：
1. 点击 Portal 首页的"CTOS Authorization"
2. 填写授权表单：
   - 姓名
   - 身份证号
   - 联系方式
   - 授权期限
3. 上传身份证复印件
4. 点击"Submit"提交
5. 等待 CTOS 报告（通常 1-2 天）

### 上传 CTOS 报告（如果已有）：
1. 点击"Upload CTOS Report"
2. 选择 PDF 文件（从 CTOS 官网下载）
3. 系统自动解析报告（AI-Powered OCR）
4. 提取关键数据：
   - 信用评分
   - 现有贷款列表
   - 月供总计
   - 信用卡信息

## 3.1 关于 DSR

**Q1: DSR 是什么？**  
A: DSR（Debt Service Ratio）是债务偿还比率，银行用来评估您是否有能力还款。公式：DSR = (总月供 / 月收入) × 100%

**Q2: 银行的 DSR 标准是多少？**  
A: 
- ≤ 55%: ✅ PASS（大部分银行会批准）
- 55-70%: ⚠️ BORDERLINE（部分银行可能批准）
- ≥ 70%: ❌ HIGH（很难批准）

**Q3: 如何降低 DSR？**  
A: 3 个方法：
1. **增加首付** - 减少贷款金额
2. **延长期限** - 降低月供（但总利息增加）
3. **先还清小额贷款** - 降低现有月供

**Q4: DSR 包含信用卡还款吗？**  
A: 不包含。DSR 只计算**贷款月供**（房贷/车贷/个贷），不包含信用卡最低还款额。

**Q5: 为什么我的 DSR 和银行算的不一样？**  
A: 可能原因：
- 收入定义不同（银行可能不算奖金/佣金）
- 现有月供遗漏（如担保贷款）
- 银行使用内部利率（vs 市场利率）

```
   - 薪资单（最近 3 个月）
   - 银行账单（最近 6 个月）
   - EPF 报表
   - 所得税报表（EA Form）

✅ 4. 使用 Top-3 推荐
   - 重点看前 3 名产品
   - 对比利率、月供、总利息

✅ 5. 加入比价篮对比
   - 至少加入 3-5 个产品
   - 并排对比细节差异

✅ 6. 考虑长期成本
   - 不只看利率，要看**总利息**
   - 30 年累计差异可达 RM 20,000+

✅ 7. 检查锁定期和罚息
   - 计划提前还款？选无罚息产品
   - 未来可能 refinance？选短锁定期

✅ 8. 保存快照分享
   - 分享给配偶/父母讨论
   - 分享给财务顾问咨询

✅ 9. 导出 PDF 留存
   - 作为决策依据
   - 申贷成功后留作记录

✅ 10. 联系银行申请
    - 选定产品后，跳转银行官网
    - 或直接致电银行贷款专员
```

## 4.2 如何优化 DSR

### 情况 1: DSR = 60%（BORDERLINE）

**问题**: 稍微超过 55% 批准线

**解决方案**:
```
选项 A: 增加首付 10%
- 原贷款: RM 500,000
- 增加首付: RM 50,000
- 新贷款: RM 450,000
- 新月供: RM 2,020.70
- 新 DSR: 52.6% ✅ PASS

选项 B: 延长至 35 年
- 原期限: 30 年
- 新期限: 35 年
- 新月供: RM 2,062.45
- 新 DSR: 53.1% ✅ PASS
- 注意: 总利息增加 RM 25,000

选项 C: 先还清车贷
- 现有车贷月供: RM 350
- 如果提前还清:
- 新现有月供: RM 2,100（只剩房贷）
- 新 DSR: 51.0% ✅ PASS
```

**建议**: 选**选项 A**（增加首付），因为：
- DSR 降至 52.6%（最健康）
- 总利息最少
- 不需要提前还贷

---

### 情况 2: DSR = 75%（HIGH）

**问题**: 远超 70% 高风险线

**解决方案**:
```
必须采取激进措施:

1. 大幅增加首付（≥30%）
   - 原贷款: RM 500,000
   - 增加首付: RM 150,000
   - 新贷款: RM 350,000
   - 新月供: RM 1,571.48
   - 新 DSR: 47.4% ✅ PASS

2. 先还清所有小额贷款
   - 还清车贷 RM 45,000
   - 还清个人贷款 RM 20,000
   - 减少月供 RM 800
   - 新 DSR: 66.8%（仍偏高，需配合首付）

3. 考虑延迟申贷
   - 继续工作 1-2 年
   - 累积更多首付
   - 同时还清小额贷款
   - 待 DSR 降至 55% 以下再申贷
```

**建议**: 如果无法大幅增加首付，**暂时不申贷**，避免被拒导致信用评分下降。

## 4.4 避免常见错误

### ❌ 错误 1: 只看利率，不看总利息
```
错误做法:
产品 A: 3.25% APR, 30 年 → 总利息 RM 283,482
产品 B: 3.45% APR, 25 年 → 总利息 RM 258,650 ✅

正确做法:
虽然产品 A 利率更低，但 30 年累计利息更多。
如果您能承受产品 B 的月供（稍高），应选产品 B。
```

### ❌ 错误 2: 忽略锁定期
```
错误做法:
选择锁定期 10 年的产品，3 年后想 refinance，
需付罚息 RM 15,000 (3% × RM 500,000)

正确做法:
如果计划 3 年内 refinance，选锁定期 ≤3 年的产品。
```

### ❌ 错误 3: 不保存快照
```
错误做法:
比较了 5 个产品，3 天后忘记哪个最好，重新比较。

正确做法:
立即保存快照，30 天内随时查看。
```

### ❌ 错误 4: 不分享给家人
```
错误做法:
独自决策，配偶不知情，事后产生分歧。

正确做法:
使用快照分享功能，全家一起讨论决策。
```

### ❌ 错误 5: DSR 刚好 55% 就申贷
```
错误做法:
DSR 55.0%，认为"刚好 PASS"就申贷。

正确做法:
留 5% 缓冲空间，目标 DSR ≤ 50%，避免：
- 银行内部标准可能更严（如 50%）
- 未来收入减少/支出增加导致超标
```

## A. DSR 快速计算

```
┌─────────────────────────────────────────────────────┐
│ DSR 快速计算公式                                     │
├─────────────────────────────────────────────────────┤
│                                                     │
│ DSR = (现有月供 + 新贷款月供) / 月收入 × 100%       │
│                                                     │
│ 批准标准:                                           │
│ ├─ ≤ 55%: ✅ PASS（银行会批准）                     │
│ ├─ 55-70%: ⚠️ BORDERLINE（需优化）                  │
│ └─ ≥ 70%: ❌ HIGH（很难批准）                       │
│                                                     │
│ 月供计算（PMT 公式）:                               │
│ i = APR / 12 / 100                                  │
│ n = 年数 × 12                                       │
│ 月供 = 本金 × i × (1+i)^n / [(1+i)^n - 1]          │
│                                                     │
└─────────────────────────────────────────────────────┘
```
