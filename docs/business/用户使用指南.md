## 系统简介

**这个系统是干什么的？**

想象你是一个财务顾问，客户拿着一堆信用卡账单、银行存款单来找你。以前你要手动一张张看，记录每笔消费，计算余额。现在这个系统帮你自动完成所有这些工作！

**核心价值：**
- ✅ 自动读取PDF账单，不用手打数据
- ✅ 自动分类消费（吃饭、购物、加油...）
- ✅ 100%准确，每一笔都对得上
- ✅ 帮客户找到最适合的贷款产品
- ✅ 自动生成专业报告，直接给客户看

### 4️⃣ 智能贷款匹配器（CTOS+DSR计算）

#### 🤔 这是什么？
帮客户找到最适合的贷款产品。像一个智能贷款顾问，自动计算客户的还款能力，推荐合适的银行和利率。

#### 💡 为什么需要？
客户想借贷款（买房、买车、生意周转），你需要：
1. 知道客户现在有多少债务（从CTOS报告看）
2. 计算客户的还款能力（DSR = 债务/收入）
3. 找到愿意批贷款的银行（不是每家银行都批）

#### 📚 关键概念

**CTOS报告是什么？**
- 马来西亚的信用报告
- 记录客户所有的贷款、信用卡债务
- 每月还款金额
- 信用评分

**DSR是什么？**
- Debt Service Ratio（债务偿还比率）
- 公式：DSR = 每月还款总额 ÷ 月收入 × 100%
- 例如：月收入RM 10,000，每月还款RM 6,000，DSR = 60%
- DSR越低越好，银行更容易批贷款

**银行的DSR要求：**
- Maybank：DSR ≤ 70%
- Public Bank：DSR ≤ 60%
- CIMB：DSR ≤ 65%
- 如果超过，银行不批！

#### 🚀 怎么使用？

**步骤1：准备CTOS报告**
1. 客户去CTOS网站下载信用报告（PDF格式）
2. 报告里有客户所有的贷款记录

**步骤2：上传到系统**
1. 点击导航栏的 **"Loan Matcher"**
2. 点击 **"Upload CTOS Report"**
3. 选择客户
4. 上传CTOS PDF
5. 输入客户月收入：RM 10,000
6. 输入想借的金额：RM 500,000（买房）
7. 点击 **"分析"**

**步骤3：系统自动分析**
系统会自动：
- ✅ 读取CTOS PDF，提取所有贷款记录
- ✅ 计算每月还款总额（例如：RM 4,500）
- ✅ 计算DSR：4,500 ÷ 10,000 = 45%
- ✅ 计算新贷款后的DSR（加上新贷款的每月还款）
- ✅ 匹配符合条件的银行产品

**步骤4：查看推荐结果**
系统显示：
```
✅ 符合条件的贷款产品（3个）

1. Maybank Home Loan
   - 利率：3.5%
   - 最高贷款额：RM 600,000
   - 需要DSR ≤ 70%
   - 你的DSR：55%（加上新贷款后）
   - 状态：✅ 符合条件

2. Public Bank Housing Loan
   - 利率：3.3%
   - 最高贷款额：RM 500,000
   - 需要DSR ≤ 60%
   - 你的DSR：55%
   - 状态：✅ 符合条件

3. CIMB FlexiHome
   - 利率：3.8%
   - 最高贷款额：RM 550,000
   - 需要DSR ≤ 65%
   - 你的DSR：55%
   - 状态：✅ 符合条件

❌ 不符合条件的产品（1个）

4. Hong Leong Bank
   - 利率：3.2%
   - 需要DSR ≤ 50%
   - 你的DSR：55%
   - 状态：❌ DSR超标
```

**步骤5：导出报告给客户**
1. 点击 **"生成报告"**
2. 系统生成专业PDF
3. 包含：
   - 客户当前债务明细
   - DSR计算过程
   - 推荐贷款产品对比表
   - 每个产品的优缺点
4. 直接发给客户，专业又清晰

#### 🎯 实际例子

**场景：**
客户YEO CHEE WANG想买房，需要借RM 500,000。

**使用系统：**
1. 上传客户的CTOS报告
2. 系统自动提取：
   - 现有车贷：每月还RM 1,200
   - 现有信用卡债务：每月最低还款RM 800
   - 个人贷款：每月还RM 1,500
   - **总计：每月还RM 3,500**
3. 输入月收入：RM 12,000
4. 当前DSR = 3,500 ÷ 12,000 = 29.17%（很健康！）
5. 新房贷每月还款约RM 2,500（假设）
6. 新DSR = (3,500 + 2,500) ÷ 12,000 = 50%
7. 系统推荐：Maybank、Public Bank、CIMB都可以批

**结果：**
客户看到报告，知道自己能借到钱，也知道哪家银行利率最低，很放心！

### 5️⃣ AI高级分析系统

#### 🤔 这是什么？
像一个智能财务分析师，自动分析客户的消费习惯、财务健康度，给出专业建议。

#### 💡 为什么需要？
你不可能每个客户都花几小时分析数据。这个AI系统自动帮你：
- 找出异常消费（例如：这个月突然多花了RM 5,000）
- 预测未来现金流（接下来3个月会花多少钱）
- 给客户评级（银牌、金牌、铂金客户）
- 推荐省钱方法

#### 🧠 AI分析项目

**1. 财务健康评分（0-100分）**

系统自动计算：
- ✅ 信用卡使用率（用了多少额度）
- ✅ 按时还款记录
- ✅ 储蓄率（收入-支出）
- ✅ 债务比率（DSR）

**评分标准：**
- 90-100分：优秀（绿色）
- 70-89分：良好（蓝色）
- 50-69分：一般（黄色）
- 0-49分：需改善（红色）

**2. 现金流预测**

系统分析过去6个月的数据，预测未来3/6/12个月：
- 📊 每月预计消费
- 📈 账户余额趋势
- ⚠️ 可能出现的资金缺口

**例子：**
```
未来3个月预测：
- 2月：预计消费RM 4,500，余额RM 12,000
- 3月：预计消费RM 4,800（农历新年），余额RM 10,500
- 4月：预计消费RM 4,200，余额RM 11,000
⚠️ 警告：3月消费可能超支，建议提前准备
```

**3. 客户分层系统**

系统自动给客户评级：
- 🥈 银牌客户：月收入< RM 5,000
- 🥇 金牌客户：月收入RM 5,000-15,000
- 💎 铂金客户：月收入> RM 15,000

**不同等级的服务：**
- 银牌：基础账单处理
- 金牌：+ AI分析 + 月度报告
- 铂金：+ 专属理财建议 + 优先贷款审批

**4. 异常消费检测**

系统自动发现不正常的消费：
- ⚠️ 单笔消费超过平均值3倍
- ⚠️ 某个类别突然暴增（例如：购物平时RM 500，这个月RM 5,000）
- ⚠️ 深夜消费（凌晨1-6点，可能是盗刷）
- ⚠️ 海外消费突然出现

**实时警报：**
系统发现异常后，自动通知你：
"⚠️ 检测到异常：客户CHEOK JUN YOON在1月15日消费RM 8,500（购物），超过平均值5倍，请检查是否正常。"

系统每天自动分析所有客户数据，你只需要：

### 7️⃣ 付款提醒

#### 🤔 这是什么？
自动提醒客户信用卡到期要还款，避免逾期。

#### 💡 为什么需要？
客户忙，会忘记还信用卡。逾期会：
- 产生罚息（通常15-18%年利率）
- 影响信用评分
- 客户不开心，怪你没提醒

这个功能自动发提醒，客户满意，你也安心。

#### 📅 提醒时间

**3次提醒：**
1. **提前7天**：友好提醒
   - "您的Maybank信用卡将在1月25日到期，需还款RM 1,234.56"
2. **提前3天**：重要提醒
   - "⚠️ 紧急：您的信用卡3天后到期，请尽快还款"
3. **到期当天**：最后通知
   - "🚨 今天是最后一天！请立即还款避免罚息"

#### 🚀 怎么使用？

**步骤1：设置提醒**
1. 上传信用卡账单后，系统自动创建提醒
2. 或手动添加：
   - 进入客户仪表板
   - 点击 **"Reminders"**
   - 点击 **"添加提醒"**
   - 选择账单、设置提醒日期
   - 保存

**步骤2：自动发送**
系统每天自动检查：
- 今天有哪些提醒要发
- 发送邮件给客户（如果设置了邮箱）
- 在系统里显示通知

**步骤3：标记已还款**
客户还款后：
1. 进入 **"Reminders"** 页面
2. 找到对应提醒
3. 点击 **"标记已还款"**
4. 系统停止发送后续提醒

## 常见问题

### ❓ 1. 为什么要按这个顺序使用？

**推荐顺序：**
```
客户资料 → 信用卡账单 → 储蓄账单 → 收据 → CTOS → 自动分析
```

**原因：**
- 信用卡账单是基础，其他功能依赖它
- 储蓄账单帮助了解现金流
- 收据匹配需要先有信用卡交易
- CTOS分析需要知道客户收入和债务
- AI分析需要足够的数据才准确

### ❓ 2. 可以跳过某些步骤吗？

**可以！**
- 如果客户只需要账单处理，上传信用卡账单就够了
- 如果客户要申请贷款，必须上传CTOS报告
- 收据是可选的，看客户需求

### ❓ 3. 数据会重复吗？

**不会！**
- 系统有防重复机制
- 同一张卡、同一个月的账单，只能上传一次
- 如果重复上传，系统会提示

### ❓ 4. 数据准确吗？

**100%准确！**
- 系统会验证每笔交易总和 = 账单总额
- 如果不对，会警告你
- 双重验证机制确保零错误

### ❓ 5. 支持所有银行吗？

**15家主流银行：**
- ✅ 信用卡：15家
- ✅ 储蓄账户：7家
- ❌ 其他银行：暂不支持（可以联系开发）

### ❓ 6. 文件会丢失吗？

**不会！**
- 所有文件自动归档，4层结构
- 客户 → 类别 → 银行 → 月份
- 随时可以找回原始PDF

### ❓ 7. 如何删除错误上传的账单？

1. 进入客户仪表板
2. 找到该账单
3. 点击 **"删除"**
4. 确认删除
5. 系统会同时删除：
   - 数据库记录
   - 关联交易
   - PDF文件

### ❓ 8. 可以导出数据吗？

**可以！**
- Excel：点击 **"导出Excel"**，包含所有交易
- CSV：点击 **"导出CSV"**，可导入其他系统
- PDF报告：自动生成，包含图表和分析

### ❓ 9. AI分析准确吗？

**准确度：**
- 基于真实历史数据
- 数据越多越准确（建议至少3个月）
- 预测准确率约80-85%

### ❓ 10. 如何获得最佳使用体验？

**最佳实践：**
1. ✅ 每月定期上传账单（不要拖）
2. ✅ 及时标记已还款
3. ✅ 上传至少3-6个月的历史数据
4. ✅ 鼓励客户也上传收据
5. ✅ 定期查看AI分析建议
