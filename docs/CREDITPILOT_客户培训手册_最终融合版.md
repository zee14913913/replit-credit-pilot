2. 选择PDF或Excel文件
3. 系统自动解析交易明细
4. 双重验证确保准确度

#### **步骤2: 查看交易列表**


#### **步骤3: 查看交易分类**

每笔交易自动分类为：
- 🏷️ **INFINITE供应商**（7家指定供应商，自动收取1%服务费）

- ✅ **已匹配** - 已找到对应收据
- ⏳ **待匹配** - 有候选收据，需确认

**功能**：批量上传收据，OCR自动提取信息，智能匹配建议。

## 第四部分：常见问题FAQ（10分钟）

### Q1: 如何提高收据匹配率？
**A**: 
1. 上传收据时确保图片清晰
2. 尽量在交易发生后7天内上传
3. 手动确认待匹配收据

### Q2: 什么是OWNER vs INFINITE分类？
**A**:
- **OWNER**: 客户个人消费和还款
- **INFINITE**: 7家指定供应商的消费和还款（自动收取1%服务费）

### Q3: 如何达到A级数据完整性评分？
**A**:
- 收据匹配率>90%
- 所有账单验证通过
- 所有交易已分类

### Q4: 支持哪些银行？
**A**:
- **信用卡**: 15家马来西亚主流银行
- **储蓄账户**: 7家银行

### Q5: 数据安全如何保障？
**A**:
- SHA-256密码加密
- 双重验证系统
- 完整审计日志

## 第五部分：最佳实践（10分钟）

### 5.1 每月工作流建议

**月初（1-5号）**：
1. 收集上月所有信用卡账单PDF
2. 批量上传到Credit Cards → Tab 1

**月中（10-15号）**：
1. 整理上月所有收据JPG/PNG
2. 批量上传到Credit Cards → Tab 2
3. 确认自动匹配建议

**月底（25-30号）**：
1. 查看Credit Cards → Tab 3供应商发票
2. 生成Credit Cards → Tab 4月结报告
3. 导出PDF/Excel给会计师

### 5.2 效率提升技巧

**技巧1: 批量上传**
- 一次性上传所有账单和收据，不要一张一张传

**技巧2: 利用4-Tab设计**
- 所有操作在Credit Cards一个模块完成，避免切换页面

**技巧3: 定期检查数据完整性评分**
- 目标：每月达到A级评分

### 5.3 评级标准参考

| 评级 | 描述 | 达成条件 |
|------|------|----------|
| A级 | 财务数据优秀 | 收据匹配率>90%，所有账单验证通过 |
| B级 | 财务状况良好 | 收据匹配率70-90%，账单数据完整 |
| C级 | 需要改进 | 收据匹配率50-70%，或存在未分类交易 |
| D级 | 数据不足 | 收据匹配率<50%，或关键数据缺失 |
