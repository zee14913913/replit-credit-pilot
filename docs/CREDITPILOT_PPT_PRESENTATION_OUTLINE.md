## 📋 演示技巧

### **开场（5 分钟）**
1. 自我介绍 + 公司背景
2. 抛出问题："您每月花多少时间管理财务？"
3. 引出痛点 + 市场机会

### **产品演示（20 分钟）**
1. **先演示结果** - 展示 Top-3 推荐和月度报告
2. **再讲过程** - OCR 识别、DSR 计算、智能匹配
3. **强调价值** - 每个功能都说"为客户节省 XX 时间/金钱"
4. **使用真实数据** - 不要用假数据，用脱敏的真实案例

### **商业模式（10 分钟）**
1. 清晰的收入来源
2. 可信的增长预测
3. 明确的盈利路径

### **总结与号召（5 分钟）**
1. 重申核心价值
2. 强调紧迫性（"为什么是现在"）
3. 明确下一步行动
