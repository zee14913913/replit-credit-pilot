**Slide 20: DSR计算器演示**
```
展示:
• 输入: 月收入/现有月供/贷款金额
• 计算: 新月供/总月供/DSR%
• 结果: ✅PASS / ⚠️BORDERLINE / ❌HIGH
• 建议: 优化方案（增加首付/延长期限）

公式:
DSR = (现有月供 + 新贷款月供) / 月收入 × 100%
月供 = PMT(本金, APR, 年数)
```

**Slide 21: Top-3智能推荐**
```
排序算法:
1. DSR状态（PASS优先）
2. 月供金额（低到高）
3. DSR百分比（低到高）

展示:
👑 #1 - Maybank FlexiRate (3.25% APR, 月供最低)
🥈 #2 - CIMB Islamic (3.35% APR)
```
