## 🔧 修复方案

### 之前的设置 ❌（太暗）

```jsx
// 视频设置
opacity: 0.3 (30%)          ← 太透明
brightness: 0.4 (40%)       ← 太暗

// 渐变遮罩
from-black/60               ← 顶部60%黑色
via-black/40                ← 中间40%黑色
to-black                    ← 底部100%黑色

结果：视频几乎看不见 ❌
```

### 现在的设置 ✅（清晰可见）

```jsx
// 视频设置
opacity: 0.6 (60%)          ← 提升到60%（翻倍！）
brightness: 0.8 (80%)       ← 提升到80%（翻倍！）

// 轻微渐变遮罩
from-black/20               ← 顶部仅20%黑色
via-transparent             ← 中间完全透明！
to-black/40                 ← 底部40%黑色（确保按钮可读）

结果：视频清晰可见 ✅
```
