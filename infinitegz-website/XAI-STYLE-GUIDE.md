# X.AI 极简风格指南 V4.0

## 核心原则

### 1️⃣ 纯黑背景
- **主背景**: `bg-black` (#000000)
- **卡片**: `bg-zinc-950/50` (半透明深灰)
- **边框**: `border-zinc-800` (极细灰色线条)

### 2️⃣ 单一品牌色
- **唯一强调色**: Purple (`text-purple-400`, `border-purple-400`)
- 仅用于：
  - CTA按钮
  - 关键指标数字
  - Logo/品牌元素

### 3️⃣ 禁用颜色
❌ **完全移除**:
- 红色状态指示器
- 黄色警告标签
- 绿色成功标签
- 彩色背景渐变
- 彩色阴影效果

### 4️⃣ 文字系统
- **主标题**: `text-white` (纯白)
- **正文**: `text-zinc-300` (浅灰)
- **次要文字**: `text-zinc-500` (中灰)

### 5️⃣ 按钮系统
```tsx
// 主按钮（唯一使用紫色）
<button className="bg-purple-600 hover:bg-purple-500 text-white px-8 py-3">
  立即开始
</button>

// 次要按钮（纯黑白）
<button className="bg-transparent border border-zinc-700 hover:border-zinc-500 text-white px-8 py-3">
  了解更多
</button>
```

### 6️⃣ 图标系统
- 使用 Lucide React 线条图标
- 统一尺寸: `w-5 h-5` 或 `w-6 h-6`
- 颜色: `text-zinc-400` (默认灰色)
- 强调: `text-purple-400` (关键指标)

## 布局规范

### 页面结构
```
┌─────────────────────────┐
│   纯黑背景 (bg-black)   │
│                         │
│  ┌───────────────────┐  │
│  │  卡片 (bg-zinc-950/50) │
│  │  边框 (border-zinc-800) │
│  └───────────────────┘  │
│                         │
└─────────────────────────┘
```

### 间距系统
- 章节间距: `py-24`
- 卡片内边距: `p-8`
- 文字间距: `space-y-4`

## 禁止事项
1. ❌ 不使用任何状态颜色（红/黄/绿）
2. ❌ 不使用渐变背景
3. ❌ 不使用彩色阴影
4. ❌ 不使用emoji或装饰元素
5. ❌ 不使用多个品牌色

## 允许使用
1. ✅ 单一紫色作为品牌色
2. ✅ 纯黑白灰色系
3. ✅ 极细边框线条
4. ✅ Lucide专业图标
5. ✅ 极简几何形状
