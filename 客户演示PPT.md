## 幻灯片 8：核心功能 #4 - 智能贷款匹配

```
🏦 CTOS + DSR 自动计算

工作原理：
1. 上传CTOS报告
   → 自动提取所有现有债务

2. 计算当前DSR
   → 债务 ÷ 收入 × 100%

3. 计算新贷款后DSR
   → 加上新贷款的月供

4. 匹配符合条件的银行
   → 推荐3-5个最佳产品

实际案例：
客户月收入：RM 8,000
现有债务：RM 3,500/月
当前DSR：43.75% ✅
新贷款：RM 500,000
新DSR：75%
推荐：3个符合条件的银行产品
```
