| PRODUCT_NAME | 产品名称 |
| PRODUCT_TYPE | 产品类型 |
| CATEGORY | 类别（Personal/Business/Personal/Business） |

## 使用说明

此数据库可用于：
1. 产品搜索和比较
2. 基于Monthly Income和Monthly Commitment的产品匹配
3. DSR计算和资格评估
4. 产品推荐系统

详细使用说明请参考：PRODUCT_MATCHING_SYSTEM.md
