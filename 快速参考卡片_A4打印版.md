## 🔧 常见问题速查

| 问题 | 原因 | 解决方法 |
|------|------|---------|
| **上传失败** | 文件格式错误 | 用电子版PDF，不要扫描件 |
| **收据识别不准** | 照片模糊 | 重拍照片，确保光线充足 |
| **找不到客户** | 客户未建档 | DASHBOARD → 新增客户 |
| **数据输入错了** | 手动失误 | 点击"编辑"按钮修改 |
| **系统卡住不动** | 文件太大 | 等待3分钟，或刷新页面 |

## 💡 省时小技巧

### ⚡ 批量上传（一次处理10份账单）
1. 所有PDF放同一文件夹
2. 文件名用客户名开头（例：张三_Maybank.pdf）
3. 一次选择多个文件上传

### 🔍 快速搜索（秒找交易记录）
- 在搜索框输入：客户名称 / 金额 / 日期 / 商家
- 按 Enter 键立即筛选

### 📥 导出Excel（深度分析）
- 任何列表页面右上角 → 点"导出" → 选"Excel"
- 下载后可自由编辑、制作图表
