# CreditPilot – Customer User Guide

**INFINITE GZ SDN. BHD.**

**Access**: https://portal.creditpilot.digital/loans/page

---

## 1. Browse Loans

Open the portal to view the latest loan offers and bank preferences.

- **Top-3 visual cards** appear automatically.
- Click **Add to Compare** to add a product to your basket.

---

## 2. Compare Loans

At the top right, select **Compare Basket** → **Compare Page**.

You can:
- Edit loan parameters (amount, tenure, rate).
- Click **Recalculate** to update DSR and monthly payment.
- View automatic ranking (PASS → BORDERLINE → HIGH).
- Use **Save Snapshot** / **Share** / **PDF Export** buttons.

---

## 3. CTOS Authorization

Use **CTOS Form** to submit your consent and documents (IC / SSM).

The system encrypts data locally and notifies our officers for report retrieval.

---

## 4. Top-3 and Ranking PDFs

- **Top-3 PDF** – Daily weighted ranking report.
- **Compare PDF** – Includes all loan inputs and DSR results.

Both use INFINITE GZ corporate branding.

---

## 5. Tips for Best Experience

✓ Access with latest Chrome / Edge.  
✓ Allow pop-ups for PDF downloads.  
✓ All data refreshes daily at 11:00 AM (KL time).  
✓ No personal token usage for clients – shared dataset only.
